# 时间流  
主要功能也是为了防止工作/生活中需要处理的任务被遗忘。  
日常在处理事情的时候，如果没有及时记录需要处理的事情，经常会遗忘，因此需要一个软件能随时记录，并在任务完成之后，总结一下事情的轨迹信息。
